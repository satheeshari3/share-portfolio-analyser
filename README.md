Share Portfolio Management Tool

A simple Python-based tool to manage and analyze share portfolios using CSV files. Designed to calculate realized profit/loss using FIFO (First In, First Out) accounting for trades. Ideal for personal use, including tracking investments for family or small-scale portfolio management.

Features

✅ Add and track multiple stocks with their buy/sell transactions

✅ Calculate realized profit/loss for each stock using FIFO

✅ Generate clean summaries of portfolio performance

✅ Handles large CSV data efficiently

✅ Customizable for personal use 


## About

This Share Portfolio Management Tool was created to help my uncle keep track of his stock investments efficiently.
It can also be adapted for personal or family use.
